# Group Policy (GPO)

## Overview
This document describes the Group Policy Objects used in the lab.

## Implemented policies
- Password policy
- Folder redirection
- Printer deployment
- Desktop and user restrictions

## Scope
Group Policies are linked to Organizational Units (OU) to control user and
computer settings in a centralized way.

## Purpose
GPO is used to enforce security settings, standardize user environments,
and simplify administration.
