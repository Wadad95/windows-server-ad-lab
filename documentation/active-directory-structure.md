# Active Directory Structure

## Overview
This document describes the Active Directory structure used in the lab.

## Organizational Units (OU)
- Domain
  - Users
  - Computers
  - Groups
  - Servers

## Users
Users are created based on department and location.

## Groups
Security groups are used to manage access to resources such as file shares
and printers.

## Design principle
The structure follows a simple enterprise design to simplify administration
and Group Policy management.
